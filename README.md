# AK Medicine Corner - Pharmacy E-Commerce Website

A modern, responsive, production-quality pharmacy e-commerce landing page built for **AK Medicine Corner**, a retail & wholesale pharmacy in Chattogram, Bangladesh.

## Project Overview

This project is a professional healthcare e-commerce website designed to inspire trust, cleanliness, safety, and professionalism. It features a clean medical UI, mobile-first responsive design, and smooth user interactions suitable for a real pharmacy business.

**Live Demo:** [View Live Site](#)

## Features

### Design & UI/UX
- Modern healthcare color scheme (Medical Teal, Trust Blue, Health Green)
- Clean, professional, and trustworthy design
- Fully responsive (mobile-first approach)
- Smooth animations and scroll effects
- Custom loading screen
- Accessible design with ARIA labels

### Pages
- **Home Page** - Complete landing page with all sections
- **Flat Rate Medicines** - Dedicated page with search, filter, and sort functionality

### E-Commerce Features
- Shopping cart sidebar with add/remove/quantity controls
- Product search functionality
- Category filtering
- Wishlist system
- Quick view modal
- Floating WhatsApp order button
- Toast notifications
- LocalStorage persistence for cart & wishlist
- Responsive product cards with hover effects
- 20+ flat rate medicine entries with pricing

### Technical
- Pure HTML5, CSS3, and Vanilla JavaScript
- Tailwind CSS for utility-first styling
- Semantic HTML structure
- Optimized for fast loading
- Clean and maintainable code

### SEO
- Semantic HTML5 markup
- Meta tags (title, description, keywords, robots)
- Open Graph tags for social sharing
- Twitter Card meta tags
- Schema.org structured data (LocalBusiness)
- Proper heading hierarchy

## Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/AK-Medicine-Corner.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd AK-Medicine-Corner
   ```

3. **Open in browser**
   Simply open `index.html` in your web browser.

4. **For local development (optional)**
   ```bash
   # Using Python
   python -m http.server 8000

   # Using Node.js
   npx serve .
   ```

   Then visit `http://localhost:8000`

## Key Interactions

- **Cart System** - Add/remove products, quantity adjustment, total calculation
- **Wishlist** - Toggle products, persisted in localStorage
- **Search** - Real-time product search filtering
- **Quick View** - Modal with product details
- **Mobile Menu** - Slide-in navigation with overlay
- **Smooth Scroll** - Anchor link navigation
- **Back to Top** - Scroll-triggered button
- **WhatsApp Integration** - Direct ordering via WhatsApp
- **Toast Notifications** - User feedback for actions
- **Scroll Animations** - Intersection Observer-based animations
